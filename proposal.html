🦞 四、LINE「龍蝦」AI 對話機器人

買方或賣方加入 LINE 官方帳號後，與「龍蝦」AI 進行自然對話。龍蝦會扮演親切的在地朋友角色，分三階段為業務創造價值：

階段 功能 說明
Phase 1 親切打招呼 用輕鬆語氣歡迎新朋友，了解對方是要買房還是賣房
Phase 2 客服建立信任 專業回答房產問題（行情、稅費、流程），展現在地專業度
Phase 3 需求情蒐 自然引導對方透露：物件位置、期望價格、急迫度、賣屋原因

✅ 情報自動回傳：當龍蝦收集到足夠情報（位置 + 價格 + 急迫度），自動推播情報摘要到業務的 Telegram，並寫入 CRM。業務拿到情報後直接聯繫，成交率大幅提升。

📱 五、Facebook AI 發文

AI 自動產生高品質貼文內容，協助經營龍蝦粉絲頁與社團：

發文管道

管道 方式 說明
龍蝦粉絲頁 全自動 使用 Facebook 官方 Graph API 自動發佈，100% 合規安全
龍蝦社團 AI 產文 + 人工發佈 AI 產生貼文草稿 → 推播到 Telegram → 業務一鍵複製貼到社團

為什麼社團不全自動？ Facebook 自 2018 年起限制第三方 App 在社團發文。為保護貴司帳號安全，社團發文採用「AI 產文 + 人工發佈」模式。AI 負責最耗時的內容創作，業務只需花 10 秒複製貼上。

AI 產文類型

發文類型 佔比 說明
🏠 物件推薦 40% 推薦近期高價值物件，引導有興趣的人私訊
📊 市場觀察 30% 在地行情趨勢分析，展現專業度
💡 買賣小知識 20% 房產知識科普，建立社團價值
🤔 互動貼文 10% 投票/問答，提升社團互動率

✅ 品質控管：所有 AI 產生的貼文都會先推播到業務 Telegram 預覽。粉絲頁確認後自動發佈，社團複製貼上即可，全程業務掌握最終發佈權。

🔒 八、資安與隱私說明

疑慮 我們的做法
591 爬蟲會不會被封鎖？ 採用隨機間隔 + 模擬人類行為 + 備用 Proxy 機制。只讀取公開資訊，不登入帳號。
Facebook 帳號安全？ 粉絲頁使用 Facebook 官方 Graph API 發文，100% 合規。社團則由 AI 產文、業務手動發佈，完全不使用自動化操作社團，零帳號風險。
LINE 對話隱私？ 所有對話資料僅儲存在貴司 Mac Mini 本機，不上傳到第三方。Claude AI 分析後不保留對話內容。
客戶資料外洩風險？ CRM 資料存放在貴司自有的 Google Sheets，我方無法存取。系統部署在貴司設備，資料不經過我方伺服器。
API Key 安全？ 所有 API Key 以環境變數儲存，不寫死在程式碼中。貴司可隨時自行更換。
